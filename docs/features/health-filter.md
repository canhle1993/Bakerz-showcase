# Health-Based Meal Suggestion

This module helps users explore foods based on selected health conditions.

## Flow

1. User enters height and weight
2. BMI is calculated on the client side
3. User selects one or more health conditions
4. System returns matching products

## Example Health Conditions

- Cardiovascular
- Diabetes
- Hypertension
- Overweight
- Underweight
- Elevated liver enzymes
- Reduce the risk of stroke
